# Start a Conversation

You're all set! Here are a few ways to start chatting with Hermes:

## Quick Start

1. Click the **Hermes Agent** icon in the Activity Bar
2. Type your message in the chat box and press Enter
3. Hermes will respond with code, explanations, or actions

## Pro Tips

- **Select code** in the editor, right-click, and choose **Hermes: Insert Selection into Chat**
- Ask questions like "Explain this function" or "How do I refactor this?"
- Use CodeLens buttons (💬 **Ask Hermes**, 🤖 **Explain this**) above functions and files
- Hermes can read files, search the workspace, check diagnostics, and run terminal commands
