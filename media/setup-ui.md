# Open the Control Center

The **Control Center** is a webview panel in the sidebar that gives you a visual dashboard for:

- Selecting the active model
- Configuring tool permissions (auto-approve read/write/terminal)
- Testing the connection to Hermes
- Importing/exporting profiles
- Previewing the system prompt

## How to Open

Click the **Hermes Agent** icon in the Activity Bar (sidebar), then click the **Control Center** tab (below the Chat tab).

Or run **Hermes: Open Settings** from the Command Palette.
